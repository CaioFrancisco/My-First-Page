# 🎂 Receita de Bolo de Chocolate Simples

Este é um projeto de uma página web simples e elegante que apresenta uma receita de bolo de chocolate, desenvolvida para praticar conceitos fundamentais de desenvolvimento web.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando:
- **HTML5**: Estruturação de conteúdo (listas, seções, cabeçalhos).
- **CSS3**: Estilização personalizada, layout responsivo e botões com efeitos.
- **YouTube API**: Incorporação de vídeo demonstrativo.

## 📋 Funcionalidades

- Lista detalhada de ingredientes para massa e calda.
- Passo a passo numerado do modo de preparo.
- Vídeo tutorial integrado diretamente na página.
- Botão de chamada para ação (CTA) para o canal do YouTube.
- Design centralizado e otimizado para leitura.
